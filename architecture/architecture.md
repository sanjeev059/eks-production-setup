# Global Video Streaming Platform

## Problem Statement

Design and implement a production-inspired global video streaming platform capable of serving 15 million concurrent users across multiple regions.

The platform should be:

- Highly Available
- Fault Tolerant
- Scalable
- Secure
- Cost Optimized
- Infrastructure as Code
- Zero Downtime Deployment

---

## Functional Requirements

- User Authentication
- Browse Videos
- Search Videos
- Watch Videos
- Upload Videos
- Notifications
- Billing
- Recommendation Engine

---

## Non Functional Requirements

- 15 Million Concurrent Users
- Global Availability
- 99.99% Uptime
- Low Latency
- Automatic Scaling
- Disaster Recovery
- Secure Communication
